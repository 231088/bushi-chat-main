# 武士チャット メイン部分

## 概要

PHPで動作する、テキストチャットサイトです。

リポジトリにあるsocket.ioサーバー(ファイル下部にリンクあり)と通信することで、

リアルタイムチャットを実現しています。

尚、**socket.ioサーバーと通信できない場合はチャットができません**。

## 仕様技術

PHP・HTML・CSS・JavaScript

PHP 8.1.2で動作を確認しています
